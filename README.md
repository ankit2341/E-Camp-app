## About the project

This is a website where you can do e-booking of camps, events and also book the particular camp.

## User features
- Login and signup
- Googlelogin
- Explore camps
- Book camps 
- Admin Panel
- Chatbot
- Filters

## Admin features
- Add new users, camps or events
- Update users, camps or events
- Delete users, camps or events
